Uncalibrated Eye Tracking

Diagnostic tool using eye movements

Download this project as a .zip file Download this project as a tar.gz file

This repository contains the code I developed to collect meaningful eyetracking data without spatial calibration, making it possible for the first time to develop a diagnostic tool that can be used in patients with severe traumatic brain injury.

This work was done in collaboration with David Heeger (NYU Psychology and Center for Neural Science), Marisa Carrasco (NYU Psychology) and Uzma Samadani (NYU School of Medicine, and VA NY Harbor Healthcare).
